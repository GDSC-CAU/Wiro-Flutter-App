## Get Started

#### Required

- Android SDK
- Android Studio
- Flutter SDK

1. Clone this Project

```shell
git clone git@github.com:GDSC-CAU/Solution-Challenge-Team-2-Flutter-App.git
```

2. Open Project with Android Studio

3. Flutter Pub Get

```shell
flutter pub get
```

4. Buil with Android Studio & Enjoy!
